This library is a living document of the styles, patterns, and components that make up the Shubox website. As the site changes, update the library so this can be a comprehensive reference.

Elements

These are global styles like typography, colors, and buttons that are used throughout the site.

Structural Elements

There are a handful of structural elements that are frequently used throughout the site and use shorthand classes for ease-of-use:

Container: .c

When applied to a div or any other html element, it will apply the standard site margins to that element's children. You should avoid nesting containers inside other containers.

Wrapper: .w

This is a generic utility class that has no base style. It is typically used to group pieces of content.

Block: .b

The block class is typically applied blocks to achieve consistent spacing.

Grid and Grid Item: .g .gi

These classes are used in the responsive grid typographysystem heavily throughout the site.

Blocks

These components that are composed of a Jekyll includes in /src/jekyll/_includes, a Sass file in src/styles/_block which is has an API that can be used statically in templates to compose pages throughout the site.

Layouts

These are templates and styles that require more specific styles that are not reflected in this guide but should be viewed directly on the site.
